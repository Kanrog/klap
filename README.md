# Klap
**Klap** is a lightweight post-install utility for turning old laptops into dedicated Klipper control terminals.

This script automates the common "laptop-specific" tweaks required after installing Debian and KlipperScreen.

## Usage

Run this command directly on your laptop terminal:

```bash
bash <(curl -s https://raw.githubusercontent.com/Kanrog/klap/main/klap.sh)
